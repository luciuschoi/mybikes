# MyBikes - 세계 유명 오토바이/자전거 브랜드 정보 포탈 사이트 PRD

## 1. 프로젝트 개요

### 1.1 프로젝트 목적
세계 유명 오토바이 및 자전거 브랜드와 모델들에 대한 종합적인 정보를 제공하는 포탈 사이트를 구축합니다. 사용자들이 다양한 브랜드의 제품 정보, 스펙, 리뷰, 가격 등을 쉽게 찾고 비교할 수 있는 플랫폼을 제공합니다.

### 1.2 타겟 사용자
- 오토바이/자전거 애호가
- 구매를 고려하는 소비자
- 브랜드 및 모델 정보를 찾는 연구자
- 딜러 및 판매업체
- 기자 및 미디어 관계자

## 2. 기능 요구사항

### 2.1 핵심 기능

#### 2.1.1 브랜드 관리
- **브랜드 등록/수정/삭제**
  - 브랜드명, 로고, 설명, 설립연도, 국가
  - 브랜드 카테고리 (오토바이/자전거/하이브리드)
  - 브랜드 공식 웹사이트 링크
  - 브랜드 히스토리 및 특징

#### 2.1.2 모델 관리
- **모델 등록/수정/삭제**
  - 모델명, 연도, 카테고리
  - 상세 스펙 (엔진, 배기량, 무게, 크기 등)
  - 가격 정보 (MSRP, 시장가)
  - 색상 옵션
  - 연료 효율성 및 성능 지표

#### 2.1.3 검색 및 필터링
- **고급 검색 기능**
  - 브랜드별, 카테고리별, 가격대별 필터링
  - 연도별, 배기량별, 연료 타입별 검색
  - 키워드 기반 검색
  - 실시간 검색 결과

#### 2.1.4 비교 기능
- **모델 비교**
  - 최대 4개 모델 동시 비교
  - 스펙별 상세 비교 테이블
  - 가격 비교 및 차트
  - 장단점 분석

### 2.2 부가 기능

#### 2.2.1 사용자 시스템
- **회원가입/로그인**
  - 소셜 로그인 (Google, Facebook)
  - 이메일 인증
  - 프로필 관리

#### 2.2.2 리뷰 시스템
- **사용자 리뷰**
  - 별점 평가 (1-5점)
  - 상세 리뷰 작성
  - 사진 업로드
  - 리뷰 신고 기능

#### 2.2.3 즐겨찾기
- **관심 모델 저장**
  - 개인 즐겨찾기 목록
  - 비교 목록 저장
  - 알림 설정

#### 2.2.4 뉴스 및 정보
- **최신 뉴스**
  - 신제품 발표
  - 브랜드 뉴스
  - 시장 동향
  - 이벤트 정보

## 3. 기술 요구사항

### 3.1 백엔드 기술 스택
- **Ruby on Rails 8.0.2**
- **SQLite3** (개발환경) / **PostgreSQL** (운영환경)
- **Puma** 웹서버
- **Solid Cache** (캐싱)
- **Solid Queue** (백그라운드 작업)
- **Solid Cable** (실시간 통신)

### 3.2 프론트엔드 기술 스택
- **Tailwind CSS 4** (스타일링)
- **Flowbite** (UI 컴포넌트)
- **Stimulus** (JavaScript 프레임워크)
- **Turbo** (SPA 기능)
- **Importmap** (JavaScript 모듈 관리)

### 3.3 데이터베이스 설계

#### 3.3.1 주요 모델
